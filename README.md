# Electric Bus Fleet Scheduler

A data-driven scheduler and simulation engine for allocating charging resources to electric buses on long-distance routes.


# Running Locally

## 1. Clone Repository

```bash
git clone https://github.com/Shiv2576/E-Bus_simulation.git
cd electric-bus-scheduler
```

## 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate:

### Windows

```bash
venv\Scripts\activate
```

### macOS / Linux

```bash
source venv/bin/activate
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Start Application

```bash
streamlit run app.py
```

The application will open in your browser.
